# E-Commerce Analytics Dashboard

## Project Overview

This project analyzes the Olist E-Commerce dataset and presents business insights through an interactive Power BI dashboard.

The dashboard is divided into three analytical layers:

* Executive Analytics
* Customer Analytics
* Operations Analytics

The objective is to help stakeholders monitor revenue performance, customer behavior, and delivery operations through actionable KPIs and visualizations.

---

## Dashboard Pages

### Executive Analytics

Provides a high-level business overview including:

* Revenue
* Order Count
* Customer Count
* Average Order Value (AOV)
* Monthly Revenue Trends
* Category Performance
* Order Status Analysis

### Customer Analytics

Focuses on customer behavior and segmentation:

* One-Time vs Repeat Customers
* Customer Segmentation (RFM)
* Customer Lifetime Value
* Geographic Customer Analysis

### Operations Analytics

Tracks operational efficiency:

* Average Delivery Days
* Late Delivery Percentage
* Freight Cost Analysis
* State-wise Delivery Performance
* Delivery Trend Monitoring
